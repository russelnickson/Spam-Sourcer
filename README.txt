It is an android app. which prevents annoying spam sms's from entering into your inbox.

Features : 
1. It has a cloud database hosted in the amazon.
2. The spam no can be registered from any user.
3. The spam list can be updated from the cloud database periodically.
4. Based on the no.of hits, the spam's severity is also register.
